# План разработки Telegram Chat Clone

## Цель
Создать веб-чат, похожий на Telegram, с сохранением сообщений в БД.

## Что понятно сразу
- Нужен простой чат на одного пользователя — без авторизации, без WebSocket
- Фронтенд: React + TypeScript, стилизация Tailwind или CSS
- Бэкенд: REST API на Python/Go/JS, хранение сообщений
- Дизайн: повторить внешний вид Telegram по скриншоту
- Деплой на сервер

## Что требует уточнения
- Структура данных сообщения — решу сам (id, text, timestamp, is_outgoing)
- Особенности деплоя на Render — разберусь по документации

## Порядок выполнения
2. Сначала фронтенд — компоненты и вёрстка
1. Потом бэкенд — чтобы сразу было куда сохранять сообщения
3. Связать фронт с API
4. Настроить Docker для локальной разработки
5. Проверить и поправить баги
6. Задеплоить на Render.com

## Принятые решения

**FastAPI (Python)** — быстро поднять REST API, знакомый синтаксис, легко подключить БД

**PostgreSQL** — надёжная БД, плюс при проверке задания.

**Docker Compose** — чтобы одной командой поднять фронт + бэк + БД локально

**Render.com** — работает хорошо, есть бесплатный тариф

**Компонентная структура** — разобью на ChatHeader, MessageList, Message, MessageInput. Каждый компонент отвечает за своё, легко поддерживать
